# Maytex

Marketing site for Maytex, a commercial furniture logistics, manpower and project coordination company working across Ireland and the UK.

## Stack

Static site. Vanilla HTML, CSS and JavaScript. Anime.js (via CDN) for scroll reveals. No build step.

- `index.html` — single page
- `css/style.css` — design system and layout
- `js/main.js` — animations, nav behaviour, project strip
- `img/` — logo and headshot
- `Web assets/` — project photos
- `video/hero.mp4` — hero background video

## Running locally

Any static server works, for example:

```
npx serve
```

Then open the address it prints.

## Notes
